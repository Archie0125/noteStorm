# Changelog

All notable changes to NoteStorm will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-01

### Added

- 視覺化白板 — 便利貼、手繪、圖層、群組
- 預設三位 AI 人設 — 蘇格拉底、艾蜜莉・狄金森、喬治・歐威爾
- 自由新增 AI 人設 — 可自訂任何角色給予建議
- 一鍵採用 — AI 建議可直接建立為新筆記
- 多選與群組 — Shift+拖曳框選，建立群組
- 圖層管理 — 多畫布、新增／切換／刪除
- 專案管理 — 新增、儲存 (.wbd)、載入
- 雙語支援 — 英文 / 繁體中文
- 隱私優先 — API Key 僅存本機瀏覽器
- 自動儲存 — 狀態儲存至瀏覽器
