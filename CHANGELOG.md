# Changelog

All notable changes to NoteStorm will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-03-01

### Added

- Visual whiteboard — Sticky notes, freehand drawing, layers, groups
- Default three AI personas — Socrates, Emily Dickinson, George Orwell
- Custom AI personas — Add any character to give advice
- One-click adopt — Turn AI suggestions into new notes
- Multi-select and groups — Shift+drag to select, create groups
- Layer management — Multiple boards, add/switch/delete
- Project management — New, save (.wbd), load
- Bilingual support — English / Traditional Chinese
- Privacy-first — API key stored locally in browser only
- Auto-save — State persisted in browser
